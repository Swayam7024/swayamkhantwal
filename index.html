About Me

ENGINEER PROFILE

I'm a BCA final-year student specializing in embedded systems, IoT, and autonomous drone technology. I design and build intelligent systems that operate independently—from drone flight controllers to real-time navigation software.

My work bridges firmware development on Pixhawk flight controllers with Python-based autonomous mission planning, sensor integration, and real-time control systems. I've built a fully autonomous drone delivery system with obstacle avoidance, GPS waypoint navigation, and FPV telemetry—combining hardware expertise with software engineering.

I'm driven to solve complex engineering problems and create systems that work reliably in real-world conditions. Currently exploring advanced applications in autonomous systems, drone technology, and embedded hardware design.
